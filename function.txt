function : 

A function is a block of code which gets executed when we call it.

syntax :

#function definition

my_function <- function(){
 statement1 
 statement2
}

#function definition in python

def my_function():
 statement1 
 statement2
 
my_function() #function call

Here my_function is a function name. It is user defined name.

function() is the keyword for defining a function.

Example :

helloFun <- function(){
  print('Hello, there!')
}

helloFun()

1) when we have function definition but no function call, we will get no output.
2) when we have function call without function definition, we will get error.
3) function name can be in smallcase, camelCase, snakecase, sentencecase

Find greater of three numbers;

findGreater <- function(p, q, r){
  if (p>q & p>r){
    return p
  }else if (q>r){
    return q
  }else{
    return r
  }
}

print(findGreater(x, y, z))

# Default Parameter :


Example :

helloFun <- function(name){
   print(paste("Hello", name))
}

helloFun("Sachin")

helloFun() 

In the above, we will error 'with no default value for "name"'

Lets change it to add default to the parameter

#######
helloFun <- function(name = 'Shyam'){
   print(paste("Hello", name))
}

helloFun("Sachin")

helloFun() 

Now, the program will execute with no error. But we will two different output

The value passed in argument will be replaced to default value, if no value passed in function call the default value will be considered.

This default value assigned to a parameter is called default parameter.

The default parameter must be assigned at the right to left.

Example :

helloFun <- function(name, name1 = 'Shyam'){
   print(paste("Hello", name, name1))
}

helloFun("Sachin", "Sehwag")

helloFun("Sachin")

Example : 

helloFun <- function(name = 'Shyam', name1){
   print(paste("Hello", name, name1))
}

helloFun("Sachin", "Sehwag")

helloFun("Sachin")

3) function recursion : When a function calls itself within its definition, it is called function recursion.


my_fun <- function(x){
  if (x >0){
    print(x)
    my_fun(x-1)
  }
}

my_fun(100)

* Function recursion is same as a loop.
