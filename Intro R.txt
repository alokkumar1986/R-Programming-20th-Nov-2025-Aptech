variable : It is a name of memory location which we store some value.

Rules : 

1) A variable name can contain alphabets (a-z, A-Z), digits (0-9), underscore(_) & period(.)

2) A variable name can start with an alphabet or period(.)
if a variable starts with a period(.), next will be a digit.

3) variable names are case sensitive. i.e. A, a are different variables

4) variable name can not contain a keyword/reserved word.
