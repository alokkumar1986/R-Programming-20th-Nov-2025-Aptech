Control Flow Statement :

1) Conditional
2) Transfer
3) Iterative

Conditional : The flow of the programming execution can be handled based on a condition.

1) if
2) if else
3) if else if

if statement :

if (condition){
  print('Hi')
}

if condition true, the print() will execute else not.

example :

x <- 10
y <- 10.0

if (x==y){
 print('X is equal to Y')
}

if else :

if (condition){
  statement
}else{
  statement1
}

if condition is true, statement will execute else statement1 will execute.

Example :

x <- 10
y <- 13

if (x>y){
 print('x is greater than y')
}else{
 print('x is smaller than y')
}

if else if :

if (condition){
 statement
}else if(condition1){
 statement1
}else{
 statement2
}


if condition is true, then only statement will be executed

if condition is false, then again it will check condition1.
if condition1 is true, then only statement1 will execute

if condition, condition1 both are false, then only statement2
execute

Example :

x <- 10
y <- 12
z <- 5

if (x>y & x>z){
 print('X is greater')
}else if (y>z){
 print('Y is greater')
}else{
 print('Z is greater')
}

Iterative :

for & while loop are called iterative statement

   print(i)
}

i <- 1  #initialisation

 print(i)
 i=i+1 #increment/decrement

Transfer : 

1) break
2) next

break : When control reads a break statement inside a loop, the control move out of the loop.

next : when control reads a next statement inside a loop, the control skips the current iteration and execute next iteration. 
 

example :

for( i in 1: 100){
   print(i)
   if(i==33){
     break
   }
}


for( i in 1: 100){
   if(i==33){
     next
   }
   print(i)
}
